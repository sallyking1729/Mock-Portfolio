The variables 'tickers' and 'bench_tickes' can be changed to differnt stocks to change the mock portfolio and the bench mark stock which the mock portfolio is being compared to 
The variable 'weights' can be changed to change the distribution of the stocks in the mock portfolio
Both the weights and tickers are indexed in the same order
The variables 'data' and 'benchdata' can be changed to alter the time period which the stocks are being analysed over. These must both be set to the same start and end date
